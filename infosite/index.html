Essentials

In a nutshell

Raisin Picker is a knowledge management tool for personal usage. Targeting knowledge workers, it is inspired by the idea of a memex. Among other features, Raisin Picker enables tag trees, relations between knowledge nodes and faceted search.

Find out more

The code is hosted on Github and is under a open source licence. Raisin Picker is based on Drupal. You require a web server with a database (e.g. MySQL) and PHP.

Features

Core functionality

Content types: Typification of content nodes (e.g. person, publication, location)
Fields: Configure fields per content type (e.g. title, body in HTML format, links, date of birth)

Tagging: Organize your content by tagging it with one or several tags.

Rating: Rate any content/node on a scale. Search and filter by rating.

Relations: Create relations between content. Categorize and descripe relations. Browse using relation links.

Content revision Create revisions of content. Browse old versions of your content.

Productivity

Reminders: Setup automatic reminders to review content after some time.

Automated import: Import content or tags using RSS standard. Setup cron to run import jobs peridically (e.g. daily, monthly).

Data export: Export any content, tag tree or relation in pure data formats.

Search

Serch index: All content of any fields is indexed and searchable.

Faceted search: Auto-complete search box; faceted search; automatic indexing

Visualization

Statistics: Run statistics and visualize in charts.

Graphs: Visualize tag structures (i.e. as a tree) or relations between nodes.

Usability

Multimedia: Attach multimedia, e.g. youtube videos, mindomo mindmaps.
Dashboard: Create and configure your dashboard showing what you need.

Bulk actions: Perform bulk actions to edit, delete or update several content items in one step.

Accessibility

Responsive design: Clean, simple, responsive design to support desktops, notebooks, tablets and smartphone displays.

Readable URLs: Define logic for short and readable URLs.

Multi-user support: Full log of user activity. Define your roles to grant to users. Define access rights for your roles.

Specification

Implementation

Raisin Picker is based on open source content management system Drupal, version 7.x. Several contributed modules (all open source) are required. Raisin Picker is an installation profile which can be selected in the Drupal installation process. It also includes theme.

Search engine

Raisin Picker works best with the open-source search framework Apache solr and its related Drupal modules.

Theme

Raisin Picker includes a theme which is a subtheme of the great Omega theme (3.x).

Sample content

Drupal modules

Following contributed modules are in use (list is not complete):

  • Hierarchical Select: Used for structured selection of tags.
  • Taxonomy Manager: Create, structure and modify tags efficiently.
  • Views (plus Views Bulk Operations): Query, filter and sort any content.
  • Panels: Create dashboards.
  • Feeds (plus several plugins): Import of data/content in different formats, e.g. CSV or JSON
  • Relation: Create relations between nodes
  • CK Editor: wysiwyg editor (what-you-see-is-what-you-get)
  • Pathauto: Clear and readable URLs.
  • Link and Date: Fields to store links and dates.
  • Rules: Among other things, with this module automatic reminder of newly added content are set.
